### How to capture an image from Kubuntu

1. <alt>+<f2>
2. type `spectacle`
3. area:  rectangular region
4. button:  Take a new screensho
5. rectangle it
6. save as: File name,  this will store the file in ~/Pictures directory.
7. Continue taking a new snapshot
